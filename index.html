Installation Instructions

  1. First, download and install Node.js using the green download button above
  2. Then download the UniMatch AI application using the purple download button
  3. Locate the downloaded UniMatch_AI_Setup.exe file
  4. Double-click the file to begin installation
  5. Follow the on-screen instructions to complete installation
  6. Launch the application and start finding your ideal university!

Application Runtime Notice

When you open UniMatch AI, a command prompt window will appear showing the local server running on port 5000. Please do NOT close this window while using the application.

* Serving Flask app 'UniMatch_AI'
* Environment: production
* Debug mode: off
* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)

This window represents the local server that powers the application. Closing it will shut down the application. The server will run on http://localhost:5000 or http://127.0.0.1:5000.

When you're done using the application, you can properly close it using the exit option within the application interface, which will automatically close the server window.
